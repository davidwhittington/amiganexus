SOFTWARE ARCHIVES
Aminet
aminet.net  /  THE PRIMARY AMIGA REPOSITORY
ESSENTIAL

Aminet is the world's largest archive of Amiga software — over 40,000 files covering games, utilities, demos, music, graphics, libraries, and developer tools. Continuously updated since 1992 and still receiving new uploads today. If it was ever made for the Amiga, it's probably here.

The Internet Archive hosts a massive collection of Amiga disk images, magazine scans, game manuals, and software. The Amiga collection includes curated sets of ADF images playable directly in-browser via browser-based UAE emulation.

THE DEMO SCENE
VAULT:1/archives/demos> WHAT WERE AMIGA DEMOS?
— Real-time audiovisual programs pushing hardware to its absolute limit.
— No pre-rendered content. Everything computed live, in sync with music.
— The Amiga's custom chips (Agnus/Blitter, Denise, Paula) made effects
possible that were technically impossible on any other consumer hardware.
— Copper effects, raster bars, full-screen parallax scrollers, sampled audio.
— The demo scene gave birth to countless professional game developers.
Pouet.net
THE DEMO SCENE DATABASE

Pouet is the primary database of demo scene productions. Filter by platform (Amiga OCS, AGA), party, and group. Watch productions, read comments, track release history. Essential for scene archaeology.

WARP TO POUET
Demozoo
SCENE ARCHIVE

Demozoo is Pouet's more structured companion — detailed entries on groups, sceners, parties, and productions with curated metadata. Better for research; Pouet better for community discussion.

AMIGA MAGAZINES
Amiga Format
UK  /  1989–2000

The highest-circulation Amiga magazine. Came with coverdisks packed with demos, utilities, and game demos. Full archive available via Internet Archive and Amiga Magazine Rack.

WARP — AMR
CU Amiga
UK  /  1989–1998

Amiga Format's primary rival. Known for aggressive review scores and strong demo scene coverage. CU's "AMOS" tutorials introduced thousands to Amiga programming.

Amiga Power
UK  /  1991–1996

Cult magazine with strong editorial voice, ruthless review honesty, and a personality that set it apart from every competitor. Still discussed affectionately decades later.

HISTORY TIMELINE
1982
Jay Miner begins the "Lorraine" project — a new computer built around a custom chip set to enable true multitasking multimedia. The Amiga concept is born.
1985
Amiga 1000 launches. Preemptive multitasking OS, custom graphics and audio chips, 4096-colour HAM mode, hardware sprites. Nothing else came close.
1987
Amiga 500 launches at consumer price point. Becomes the best-selling Amiga model. Kickstart 1.3 ROM. The machine that defined a generation of gamers in Europe.
1992
Amiga 1200 and Amiga 4000 launch with the AGA chipset — 256 colours on screen (from 16.7M), improved blitter, 32-bit CPU. The peak of classic Amiga development.
1994
Commodore declares bankruptcy. Amiga IP enters legal limbo. The community refuses to stop developing.
2001
Hyperion Entertainment licenses AmigaOS. AmigaOS 4.x development begins for PowerPC hardware. The OS lives on.
2021
AmigaOS 3.2 released by Hyperion — the first major classic OS update in decades. New features, 64-bit FAT, USB stack, and critical bug fixes. Still under active development.
NOW
The Amiga community is active and growing. New hardware ships. New software is written. New demos are released. The Nexus exists to keep it all connected.
