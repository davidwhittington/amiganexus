LIVE FEED — TRANSMISSIONS UPDATED MANUALLY IN PHASE 1
2026.02.18 FPGA
MiSTer Amiga Core v1.6 Released — AGA Improvements
The MiSTer Amiga (Minimig) core has received a significant update with improved AGA compatibility, fixed HAM8 rendering artefacts, and better timing accuracy for ECS demoscene productions. The update also introduces experimental support for the CD32 FMV module in emulated form. Update via the MiSTer Downloader script or manually from the GitHub releases page.
2026.01.30 HARDWARE
Checkmate A1500 Mini — New Production Batch Open
The Checkmate A1500 Mini has reopened orders for a new production batch. This run includes an updated PCB revision with improved grounding for the A1200 motherboard variant and a new side panel option. Orders ship Q2 2026. Check checkmate1500.com for pricing and delivery estimates.
2026.01.14 OS UPDATE
AmigaOS 3.2.3 Patch Released by Hyperion
Hyperion Entertainment has shipped AmigaOS 3.2.3, a maintenance patch addressing memory handling issues in the 64-bit FAT driver, a fix for a rare crash in Intuition when closing windows with active gadgets, and an updated AmigaGuide reader. Existing 3.2 or 3.2.2 users can apply the patch; new installations should use the full 3.2.3 release. Available from Hyperion directly and through authorised resellers including AmigaKit.
2025.12.21 FPGA
Vampire V4 SA+ Firmware 2.14 — SAGA Graphics Enhancements
The Apollo Team has released Vampire V4 SA+ firmware 2.14. This update brings refined SAGA chunky modes with improved palette accuracy, a faster blitter path for 2D operations, and fixes for a timing regression introduced in 2.12 affecting some WHDLoad titles. The update is available via the Apollo Update Tool in Apollo OS.
2025.12.06 SCENE
Revision 2026 Demo Compo Entries Open — Amiga Category Returns
Revision 2026 (April, Saarbrücken) has confirmed the Amiga OCS and AGA demo categories will run as separate compos this year following strong turnout in 2025. Entry submission opens January 2026 via the Revision party website. The party will also run a dedicated Amiga hardware exhibition area.
2025.11.19 HARDWARE
AmigaKit Restocks A1200 Keyboard PCBs
AmigaKit has restocked the A1200 keyboard replacement PCB — a modern reproduction that solves the notorious membrane failure issue on aging A1200 units. Compatible with original keycaps. Also back in stock: A500 power supply replacements and A1200 trapdoor RAM expansion boards.
2025.10.31 COMMUNITY
English Amiga Board Passes 250,000 Posts
The English Amiga Board — the world's largest English-language Amiga forum — has surpassed 250,000 posts across all sub-forums. The milestone reflects the continued health of the community. EAB moderators report a 20% increase in new registrations in 2025 compared to 2024, driven partly by FPGA and emulation interest from a new generation of users discovering the platform.
2025.10.04 HARDWARE
Individual Computers Ships ACA1234 Rev 2 — 128MB RAM Support
Jens Schönfeld's Individual Computers has shipped the ACA1234 Rev 2 accelerator for the Amiga 1200. The revision doubles maximum RAM to 128MB Fast via a revised memory controller, adds a real-time clock, and improves compatibility with Zorro expansion via the ACA1234's optional carrier board. Available now from icomp.de and authorised retailers.
