WINUAE — THE GOLD STANDARD
WinUAE
WINDOWS ONLY  /  TONI WILEN

The most accurate and fully-featured Amiga emulator in existence. WinUAE emulates every Amiga model from A1000 to A4000, all chip sets (OCS/ECS/AGA/AAA), all CPU variants, and an enormous range of expansion hardware.

The learning curve is steep — the configuration dialog has hundreds of options — but no other emulator matches its depth or accuracy.

FS-UAE
WINDOWS / macOS / LINUX  /  FRODE SOLHEIM

Built on the WinUAE/UAE core but wrapped in a modern, cross-platform launcher. FS-UAE Launcher includes a built-in game database, community configuration sharing, and a cleaner setup experience than raw WinUAE.

The best choice for most users who aren't on Windows-only workflows.

macOS & RASPBERRY PI
vAmiga
macOS NATIVE  /  DIRK HOFFMANN

A ground-up native macOS Amiga emulator — not a port. Clean SwiftUI interface, built-in debugger, cycle-accurate emulation. Focuses on A500 and A500+ accuracy. Excellent for Mac users who want a first-class native experience.

WARP
AmiBerry
RASPBERRY PI / ARM  /  MICK GULLEY

Optimised UAE derivative for Raspberry Pi and ARM boards. Runs full-speed on Pi 4 and Pi 5 with HDMI output. The fastest path to a dedicated standalone Amiga box using commodity hardware. Pre-built RetroPie images include AmiBerry.

Best Pi models: Pi 4 (4GB) or Pi 5 for best performance.

WHDLOAD — GAMES FROM HDD
WHDLoad
whdload.de  /  INSTALL GAMES TO HARD DISK

WHDLoad installs floppy-based Amiga games and demos to hard disk or SD card, eliminating the need to swap floppies and often fixing compatibility issues with accelerated machines. Over 3,000 titles are supported. An essential companion to any serious Amiga emulation setup.

Each game gets a "slave" install file. Point WHDLoad at the slave, press play.
